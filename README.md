Как найти ответ на задачу комиваяжёра?
--
Задача комиваяжёра расположена в `test/tsp`.
Для поиска ответа на эту задачу нужно добавить её в тест.
1. Добавить папку в `test/tsp/tests` в формате `testtN`, где $N$ -- следующий по порядку номер.
2. Добавить в эту папку файл `graph.csv` с графом.
3. Увеличить число итераций в цикле `test/tsp/tests/__init__.py`, чтобы их количество совпало с количеством тестов.
